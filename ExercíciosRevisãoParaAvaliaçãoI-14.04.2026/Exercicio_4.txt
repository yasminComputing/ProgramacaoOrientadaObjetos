4. Qual a diferença entre os tipos de dados String, int e boolean? Dê um exemplo de uso de cada um.

 O String é  utilizado para armazenar sequências de caracteres (texto), podendo incluir letras, números, símbolos e espaços:

            Exemplo: String nome = "João";
                     String dataNascimento = "23/02/2000";

O int serve para armanzear número inteiros,positivos ou negativos:

            Exemplo: int idade = 20;

O boolean é usado para lógica, retornando apenas dois valores possíveis true (verdadeiro) ou false (falso):

            Exemplo: 
                    boolean encontrado = false;
                     int[] vetor = {1,2};
                     
                     for(int i = 0; i<2;i++){
                        if(vetor[i] == 2){
                            encontrado = true;
                        }
                     }
                     if(encontrado){
                        System.out.println("Número 2 encontrado no vetor");
                     }
